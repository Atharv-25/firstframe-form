# Firstframe × Dream Beauty — Creator Onboarding Form

A clean, elegant creator registration form that collects creator details and sends them directly to Google Sheets.

## Features
- **Firstframe × Dream Beauty** branded header
- Collects: Name, Email, Phone, City, Address (for product delivery), Instagram, Follower Count, Content Niche
- Google Sheets integration via Apps Script
- Glassmorphism design with floating particle animations
- Fully responsive

## Setup
1. Deploy `google-apps-script.js` as a Google Apps Script Web App
2. Paste the Web App URL into `script.js` (line 8)
3. Open `index.html` in a browser or deploy to any static host (Vercel, Netlify, GitHub Pages)
